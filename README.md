🏆 Student Olympic Games Website








A modern school sports competition website designed to manage and showcase Olympic-style student events.

The platform allows students to:

View sports events

Explore competition details

Register for sports activities

Track event schedules

The design uses a modern dark UI with glassmorphism and bento-grid layouts, inspired by modern startup landing pages.

⚙️ How It Works
Dynamic Event Rendering

The homepage uses JavaScript to generate sports cards dynamically.

Example events:

Varsity Sprints

Shot Put

Long Jump

3v3 Basketball

Relay Finals

Obstacle Course

Tug of War

Each event gets a randomized competition date and time.

🚀 Deployment

You can deploy this website easily using GitHub Pages.

Steps

Upload files to a GitHub repository

Go to Repository Settings

Open Pages

Select:

Folder: /root

Save
