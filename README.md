# 🏆 Student Olympic Games Website





A modern **school sports competition website** designed to manage and showcase Olympic-style student events.

The platform allows students to:

- View sports events
- Explore competition details
- Register for sports activities
- Track event schedules

The design uses a **modern dark UI with glassmorphism and bento-grid layouts**, inspired by modern startup landing pages.

# ⚙️ How It Works

The homepage dynamically generates sports event cards using JavaScript.

Example events include:

- Varsity Sprints
- Shot Put
- Long Jump
- 3v3 Basketball
- Relay Finals
- Obstacle Course
- Tug of War

Each event card displays a **randomized competition date and time**.

# 🚀 Deployment

You can deploy this website easily using **GitHub Pages**.

### Steps

1. Upload the project files to a GitHub repository
2. Open **Repository Settings**
3. Go to **Pages**
4. Select:

```
Branch: main
Folder: /root
```

5. Save

Your website will be available at:

```
https://USERNAME.github.io/REPOSITORY
```
